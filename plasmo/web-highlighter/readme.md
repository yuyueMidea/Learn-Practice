📦 项目总结

1️⃣ 网页高亮工具

一个智能的文本高亮扩展，支持：
- 🎨 6种预设颜色选择
- 💾 自动保存高亮内容
- 🔍 按页面管理高亮
- 📊 查看所有高亮统计
- ♻️ 刷新页面后自动恢复高亮

核心技术点：
- Content Script 实现 DOM 操作
- Storage API 持久化数据
- React Hooks 管理状态
- 消息通信实现跨组件交互

启动指南：
```
# 创建项目
pnpm create plasmo web-highlighter
cd web-highlighter

# 安装依赖
pnpm install
pnpm install @plasmohq/storage @plasmohq/messaging
pnpm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

运行项目:
```
# 开发模式
pnpm dev

# 构建生产版本
pnpm build
```

加载到浏览器
- 打开 Chrome，访问 chrome://extensions/
- 启用"开发者模式"
- 点击"加载已解压的扩展程序"
- 选择 build/chrome-mv3-dev 文件夹

### 🖍️ 网页高亮工具使用指南

第一次使用

1. 找到扩展图标:加载完扩展后，在浏览器右上角会看到一个 🖍️ 图标（或者你自定义的图标）。

如果没看到图标：
- 点击浏览器右上角的 拼图图标 🧩（扩展管理）
- 找到"智能网页高亮工具"
- 点击 📌 固定，图标就会出现在工具栏

2. 打开控制面板:点击 🖍️ 图标，会弹出一个漂亮的面板：
```
┌────────────────────────────────┐
│  🖍️ 智能高亮工具              │
│  选择颜色，然后选中文本进行高亮 │
├────────────────────────────────┤
│  [颜色选择区域]                │
│  🟡 🟢 🔵 🟣 🟠 🩷           │
├────────────────────────────────┤
│  [ 启用高亮模式 ]              │
├────────────────────────────────┤
│  当前页面高亮 (0 条)           │
└────────────────────────────────┘
```

### 基础操作：高亮文本

**步骤 1：选择颜色**

在弹出的面板中，点击你喜欢的颜色：
```
🟡 黄色 - 最常用，醒目
🟢 绿色 - 适合标记正确答案
🔵 蓝色 - 适合标记重要信息
🟣 紫色 - 适合标记关键概念
🟠 橙色 - 适合标记需要注意的
🩷 粉色 - 适合标记个人笔记
```

提示： 选中的颜色会有一个 ✓ 标记和蓝色边框。

**步骤 2：启用高亮模式**

点击 "启用高亮模式" 按钮：
- 按钮变成绿色 ✅
- 显示"高亮模式已启用"
- 网页底部会显示提示条

步骤 3：高亮文字
- 现在你可以高亮任何网页上的文字了！
- 在网页上用鼠标拖选文字（至少 2 个字）
- 松开鼠标,立即看到效果：
   - 文字背景变成你选的颜色 🎨
   - 右上角显示"✓ 高亮已保存"
   - 高亮自动保存，刷新页面也不会丢失
