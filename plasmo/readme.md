## Plasmo 简介

Plasmo 是一个现代化的浏览器扩展开发框架，它极大地简化了浏览器插件的开发流程。相比传统的手动配置 manifest.json 和处理复杂的构建流程，Plasmo 提供了类似 Next.js 的开发体验。

核心特点
- 开箱即用的功能：自动生成 manifest.json、热重载、TypeScript 支持、React 集成等。
- 跨浏览器支持：一套代码可以构建适用于 Chrome、Firefox、Edge、Safari 等多个浏览器的扩展。
- 文件系统路由：基于文件结构自动配置扩展的各个部分（popup、options、content scripts 等）。
- 现代化工具链：内置 Parcel 打包器，支持最新的 JavaScript/TypeScript 特性。
