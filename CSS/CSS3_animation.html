探索现代Web动画的无限可能

基础动画效果

弹跳动画

Bounce

经典的弹跳效果，常用于按钮和提示

@keyframes bounce { 0%, 20%, 53%, 80%, 100% { transform: translate3d(0,0,0); } 40%, 43% { transform: translate3d(0,-30px,0); } }

震动动画

Shake

震动效果，用于错误提示或引起注意

@keyframes shake { 0%, 100% { transform: translateX(0); } 10%, 30%, 50%, 70%, 90% { transform: translateX(-10px); } 20%, 40%, 60%, 80% { transform: translateX(10px); } }

脉冲动画

Pulse

心跳般的脉冲效果，突出重要元素

@keyframes pulse { 0% { transform: scale(1); } 50% { transform: scale(1.2); } 100% { transform: scale(1); } }

摆动动画

Swing

钟摆式摆动，优雅的自然运动

@keyframes swing { 20% { transform: rotate(15deg); } 40% { transform: rotate(-10deg); } 60% { transform: rotate(5deg); } 80% { transform: rotate(-5deg); } 100% { transform: rotate(0deg); } }

变换动画效果

旋转动画

Rotate

360度持续旋转，加载动画的经典选择

@keyframes rotate { from { transform: rotate(0deg); } to { transform: rotate(360deg); } }

翻转动画

Flip

3D翻转效果，展示卡片正反面

@keyframes flip { 0% { transform: perspective(400px) rotateY(0); } 40% { transform: perspective(400px) rotateY(-180deg); } 100% { transform: perspective(400px) rotateY(-180deg); } }

摇摆动画

Wobble

不规则摇摆，生动有趣的动画效果

@keyframes wobble { 0% { transform: translateX(0%); } 15% { transform: translateX(-25%) rotate(-5deg); } 30% { transform: translateX(20%) rotate(3deg); } }

悬浮动画

Float

轻柔的上下浮动，营造轻盈感

@keyframes float { 0%, 100% { transform: translateY(0px); } 50% { transform: translateY(-20px); } }

进入动画效果

淡入动画

Fade In

渐现效果，温和的出现方式

@keyframes fadeIn { from { opacity: 0.3; } to { opacity: 1; } }

上滑淡入

Fade Up

从下方滑入并淡现

@keyframes fadeInUp { from { opacity: 0; transform: translateY(30px); } to { opacity: 1; transform: translateY(0); } }

左滑进入

Slide Left

从左侧滑动进入

@keyframes slideInLeft { from { transform: translateX(-100px); opacity: 0; } to { transform: translateX(0); opacity: 1; } }

弹跳进入

Bounce In

弹性进入效果，活泼有趣

@keyframes bounceIn { 0% { opacity: 0; transform: scale(0.3); } 50% { opacity: 1; transform: scale(1.05); } 100% { opacity: 1; transform: scale(1); } }

彩虹色彩

Rainbow

彩虹色彩变换，绚丽夺目

@keyframes rainbow { 0% { background: linear-gradient(135deg, #ff0000, #ff8000); } 14% { background: linear-gradient(135deg, #ff8000, #ffff00); } 28% { background: linear-gradient(135deg, #ffff00, #80ff00); } }

心跳动画

❤️

心跳节奏，表达情感共鸣

光晕效果

Glow

发光效果，科技感十足

@keyframes glow { from { box-shadow: 0 0 10px #667eea; } to { box-shadow: 0 0 20px #667eea, 0 0 30px #667eea, 0 0 40px #667eea; } }
