Flexbox弹性布局

特点：现代CSS布局方案，灵活性强，支持各种对齐方式，响应式设计的首选。

弹性侧边栏

flex: 0 0 200px

不伸缩，固定宽度

弹性主内容

flex: 1 - 自动扩展

Flexbox提供了强大的对齐能力：

  • justify-content: 主轴对齐
  • align-items: 交叉轴对齐
  • flex-direction: 方向控制
  • flex-wrap: 换行控制
.flex-layout { display: flex; gap: 20px; } .flex-sidebar { flex: 0 0 200px; } .flex-main { flex: 1; }

瀑布流布局 (Masonry)

特点：使用CSS多列布局实现瀑布流效果，适合图片展示、Pinterest风格的内容布局。

项目 1

瀑布流布局让内容以瀑布的形式流动排列。

项目 2

不同高度的内容会自动填充空隙。

项目 3

这种布局特别适合展示图片或者长度不一的文本内容，营造出错落有致的视觉效果。

项目 4

Pinterest就是这种布局的典型应用。

项目 5

CSS column-count属性让实现变得简单。

项目 6

响应式设计可以调整列数以适应不同屏幕。
