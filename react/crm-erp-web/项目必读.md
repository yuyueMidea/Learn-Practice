# 📦 企业级 CRM/ERP Web 前端应用

## 🎯 项目简介

这是一款基于 React 18.3 + Tailwind CSS v3 构建的企业级 CRM/ERP 前端应用系统,提供完整的客户管理、销售管理、库存管理、采购管理等核心业务功能。

**技术栈**: React 18.3 + Vite 5 + Tailwind CSS v3 + Zustand + React Router v6

**适用场景**: 中大型企业内部办公系统、CRM 客户管理、ERP 进销存管理

## ⚡ 快速启动 (3种方式)

### 方式 1: 使用启动脚本 (推荐,最简单)

**Windows 用户**:
```
双击运行 "启动项目.bat"
```

**Mac/Linux 用户**:
```bash
./启动项目.sh
```

### 方式 2: 使用命令行

```bash
# 1. 安装依赖
npm install

# 2. 启动项目
npm run dev
```

### 方式 3: 一键启动

```bash
npm install && npm run dev
```

## 🐛 遇到问题?

### 常见问题

**Q: 依赖安装失败?**
A: 清除缓存后重新安装
```bash
npm cache clean --force
rm -rf node_modules package-lock.json
npm install
```

**Q: 端口被占用?**
A: 修改 `vite.config.js` 中的端口号

**Q: 页面空白?**
A: 查看浏览器控制台错误信息

**Q: 样式不显示?**
A: 检查 Tailwind CSS 配置
