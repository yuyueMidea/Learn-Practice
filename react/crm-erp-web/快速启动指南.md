# 企业级 CRM/ERP Web 前端应用 - 快速启动指南

## 🚀 快速开始 (3步启动项目)

### 步骤 1: 安装依赖

```bash
# 进入项目目录
cd crm-erp-web

# 安装依赖 (推荐使用 pnpm,更快)
npm install

# 或使用 pnpm
pnpm install

# 或使用 yarn
yarn install
```

**注意**: 首次安装可能需要 3-5 分钟,请耐心等待。

### 步骤 2: 启动开发服务器

```bash
npm run dev
```

启动成功后,浏览器会自动打开 `http://localhost:5173`

### 步骤 3: 登录测试

使用以下测试账号登录:

- **管理员账号**: `admin` / `admin123`
- **销售账号**: `sales` / `123456`

## 🐛 常见问题

### 1. 依赖安装失败

**问题**: `npm install` 报错

**解决方案**:
```bash
# 清除缓存
npm cache clean --force

# 删除旧的依赖
rm -rf node_modules package-lock.json

# 重新安装
npm install
```

### 2. 端口被占用

**问题**: 5173 端口已被使用

**解决方案**:
修改 `vite.config.js`:
```javascript
server: {
  port: 5174,  // 改为其他端口
}
```

### 3. 样式不生效

**问题**: Tailwind CSS 样式没有应用

**解决方案**:
检查 `tailwind.config.js` 中的 content 配置是否正确。
